Site Plan: Infinity Nikki Hub

Site Name

Name: Infinity Nikki Hub

The name directly identifies the game while "Hub" suggests a comprehensive resource center. Straightforward and SEO-friendly to make it easy for players to find.

Site Purpose

Infinity Nikki Hub serves as a comprehensive resource for players of Infinity Nikki and offers:

  • Game guides and tutorials
  • News updates and event coverage
  • Outfit compendium and showcases
  • Character profiles and lore information
  • Development history and franchise evolution

Planned Pages

The following is a list of ideas for pages that will be included (more will be added over time):

  • Home : The landing page
  • Events : Ongoing, upcoming and events that have ended can be found on this page
  • Newsletter : A form for users to subscribe to the site's newsletter so they don't miss any updates
  • News & Notices : Official announcements from the game as well as game patch notes
  • Compendium : A collection of outfits that can be collected in the game
  • Photobook : A collection of photos taken in the game

Scenarios

  1. How do I unlock specific outfits in Infinity Nikki
  2. What's the history behind the Nikki franchise and how has it evolved?
  3. What's the event that's currently ongoing and what does it offer?
  4. When's the next major update and what will it include?
  5. How do I find certain items or complete specific challenges in the game?

Color Schema

  • Primary Color (Dark slate gray: #455E61): Headers, highlights
  • Accent Color (Salmon pink: #EC9BA8): Accents, borders, important buttons
  • Background Color (Alabester: #EFE6D8): Page background
  • Element Color (Floral white: #FBF8F2): Card, section backgrounds
  • Text color (Dark gray: #333333): Main content text

Typography

  • Headers: Playfair Display - Matches the game's feminine and elegant aesthetic.
  • Body Text: Roboto - Clean and highly readable for main content
  • Accent Text: Montserrat - For navigation and special sections
