Site Plan: Wuthering Waves Chronicles

Site Name

Name: Wuthering Waves Chronicles

The name combines the game title with "Chronicles" to suggest comprehensive coverage and ongoing documentation of the game's content

Site Purpose

Wuthering Waves Chronicles will serve as a resource that will:

  • Document the open-world exploration
  • Gameplay guides and mechanics explanations
  • Share regular updates on new content and events
  • Feature story analysis and lore documentation
  • Include a community section for player discussions and contributions

Scenarios

  1. What are the optional character builds for endgame content?
  2. Where can I find specifif resources in the open world?
  3. How do I complete specific story missions and side quests?
  4. What's the actual lore and hidden lore of the game?

Color Schema

  • Primary Color (White: #FFFFFF): Headers, highlights
  • Accent Color (Ecru: #C9AC67): Accents, borders, important buttons
  • Background Color (Eerie black: #252320): Page background
  • Element Color (Charcoal: #353B4D): Card, section backgrounds
  • Text Color (Timber wolf: #D5D5D5): Main content text

Typography

  • Headers: Oxanium - Perfectly captures the sci-fi aesthetic of the game
  • Body Text: Inter - Modern and highlt readable for main content
  • Accent text: Barlow - Clean and modern for navigation and special sections
